# Modern Dental Clinic Website Template

A premium, production-ready dental clinic website template built using HTML, CSS, and JavaScript.

Designed as a reusable foundation for modern dental clinics, orthodontists, cosmetic dentists, implant specialists, and family dental practices.

---

## Features

### UI & UX

 Premium responsive design
 Modern hero section
 Before & After comparison slider
 Interactive patient testimonials
 Animated statistics
 Facilities gallery
 FAQ accordion
 Premium hover animations
 Scroll reveal animations
 Mobile-first responsive layout
